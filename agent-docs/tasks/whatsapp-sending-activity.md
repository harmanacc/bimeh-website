## WhatsApp Sending and Activity Logging

### Description

Add functionality to send messages via WhatsApp Web links and log activities.

### Steps

1. Integrate send button in message panel with confirmation modal.
2. Generate WhatsApp link with encoded message and phone.
3. After sending, log activity in database and convert lead to customer if needed.
4. Update UI to reflect sent status.
