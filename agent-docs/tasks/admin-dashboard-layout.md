## Admin Dashboard Layout

### Description

Set up the admin dashboard layout with sidebar navigation for the outreach panel, supporting RTL for Persian text.

### Steps

1. Create admin layout at `src/app/admin/layout.tsx` with sidebar.
2. Build sidebar component `src/components/admin/sidebar.tsx` with navigation items: Upload Leads, Leads Table, Customers Table, Activity History, Export Data.
3. Ensure RTL layout and Persian text support using Tailwind classes.
