## Message Generation Panel

### Description

Create the main panel for generating and customizing WhatsApp messages with AI personalization.

### Steps

1. Create message panel page at `src/app/admin/outreach/messages/page.tsx`.
2. Add global message template textarea with placeholders.
3. Implement per-user message flow: table with radio options for standard or AI customize.
4. Integrate AI API call for personalization using OpenRouter or fallback.
5. Add live preview for messages.
